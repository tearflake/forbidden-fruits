# Forbidden Fruits

This repository represents an intellectual installation, and contains some of my experiments in making programming frameworks. The experiments include relaxed BNF and implementation pseudo code.

Here they are:

* imp - simple imperative programming framework
* fnc - simple functional programming framework
* rwr - simple rewriting programming framework
* combo - a single framework combining the above three programming paradigms

Have fun exploring: [tutorial](./docs/tutorial.md)
